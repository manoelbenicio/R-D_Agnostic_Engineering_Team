    2. Testable properties:

       - rotate-before-commit: rotation only at session boundary, never mid-stream

    3. Test approach:
       - Unit test: mock L2 runtime, verify Go does not call rotation during active session
       - Integration test: start session, inject runtime events, verify no Go rotation triggered
       - Property: Go's rotate() is never called between session_started and session_stopped events
  </action>
